# Changelog

## 2026-06-22

### Multilingual entry points

- Added README language options for English, Chinese, Japanese, and Korean.
- Added `README.zh-CN.md`, `README.ja.md`, and `README.ko.md` as localized GitHub entry points.
- Updated the default README with a language switcher and multilingual update-log entry.

### Initial public release

- Introduced Agentic Development Control Plane Engineering as a lightweight method for governing multi-agent software development.
- Defined the four engineering pillars: Harness Engineering, Loop Engineering, Context Engineering, and Gate Engineering.
- Added a reference operating model for business owner, PM / gatekeeper agent, coding leader agent, module worker agents, and review agents.
- Added templates for Node Closure Pack, Next Session Bootstrap, Worker Handoff, and pull requests.
- Added prior-art notes covering GitHub Copilot coding agents, Claude Code skills and subagents, SWE-agent, SWE-bench, OpenHands, and AutoGen.
